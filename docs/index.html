正規表現のアクション

  • 受理状態にいたった場合のアクション
  • 文字クラスに関連づけられた遷移アクション
0x@A([0-9]@B|[A-F]@C|[a-f]@D)+ %E
action A {
  RV = 0
}
action B {
  RV *= 16
  RV += R - '0'
}
action C {
  RV *= 16
  RV += R - 'A' + 10
}
action D {
  RV *= 16
  RV += R - 'a' + 10
}
action E {
  save(RV)
}
