正規表現の抽象構文木

operator #operands precedence description
[ 1 1 character class
* 1 2 0 or more repetition
? 1 2 optional
. 2 3 concatenation
| 2 4 union
- 2 N/A difference
@ 2 N/A transition action
% 2 N/A final action

正規表現の遷移アクション

  • 文字クラスに関連づけられたアクションを遷移アクション (transition action) と呼ぶ。
  • 複数の遷移アクションは並列に実行される？

正規表現の受容アクション

  • 正規表現が受容された場合のアクションを受容アクション (accept action) と呼ぶ。
  • 終了状態は優先度を持つので、優先度が高い受容アクションだけが実行される。
